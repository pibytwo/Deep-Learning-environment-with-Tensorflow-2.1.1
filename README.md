3. Flask-RESTful==0.3.8
4. imutils==0.5.3
5. scikit-learn==0.22.2
6. gevent==20.5.0
7. PyMySQL==0.9.3
8. pymongo==3.10.1
9. Flask-SocketIO==4.3.0
10. requests==2.23.0

All these packages help make deployment of Deep Learning project easy. Simply pull the image, copy your code and deploy.
